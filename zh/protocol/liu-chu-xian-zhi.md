# 流出限制

Scallop 在借貸池中實施了提款限額，以維持經濟穩定性和安全性，防止大規模的突然提款可能對系統造成的不穩定。這樣可以確保流動性，支持借款人的資金使用需求，並將波動性降到最低，這對於用戶信任和平台的財務完整性至關重要。

| 資產    | 每日流出限制          |
| ----- | --------------- |
| SUI   | 3,000,000 SUI   |
| wUSDC | 3,000,000 wUSDC |
| wUSDT | 3,000,000 wUSDT |
