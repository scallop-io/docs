## **What is Scallop?**

Scallop is a DeFi game on Solana, users can stake their tokens in Scallop Pools to grow scallops in Scallop Tank, while they stake tokens, they can get the same worth of Scallop Token \(SCA\). 

At the end of each week, one or a few winner\(s\) chosen by program randomly will share out some SCA as the staking reward. During the game users can win tokens and Scallop Decorations by carrying out some tasks‚ú®!
